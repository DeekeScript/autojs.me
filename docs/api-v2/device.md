# device - 设备

设备信息相关 API。

## device.width

**功能：** 屏幕宽度（像素）

```javascript
import { device } from '@autojs/pro'

console.log("屏幕宽度：" + device.width);
```

## device.height

**功能：** 屏幕高度（像素）

```javascript
console.log("屏幕高度：" + device.height);
```

## device.model

**功能：** 设备型号

```javascript
console.log("设备型号：" + device.model);
```

## device.brand

**功能：** 设备品牌

```javascript
console.log("设备品牌：" + device.brand);
```

## device.sdkInt

**功能：** Android SDK版本

```javascript
console.log("SDK版本：" + device.sdkInt);
```

## device.release

**功能：** Android版本号

```javascript
console.log("Android版本：" + device.release);
```
