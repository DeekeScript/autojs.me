# V9

Auto.js Pro V9 是基于 Node.js 引擎的第二代 API，提供了更强大的功能和更好的性能。

## 主要特性

- 支持所有 Node.js 16.x 特性
- 可以使用 require() 导入模块
- 支持 async/await
- 支持 ES6+ 语法
- 可以使用 npm 包
- 性能提升 100 倍以上

## 快速开始

```javascript
const { app } = require('@autojs/pro');

// 启动应用
await app.launchApp('微信');

// 等待并操作
await sleep(2000);
await click(500, 1000);
```

## Node.js 特性

V9 版本完全支持 Node.js 16.x 的所有特性，包括：

- ES2021 语法支持
- 模块系统（CommonJS 和 ES Modules）
- 异步编程（Promise、async/await）
- 内置模块（fs、path、http 等）
- npm 生态系统

## 性能提升

相比第一代 API，V9 版本的性能提升了 100 倍以上，主要体现在：

- 更快的脚本执行速度
- 更高效的内存管理
- 更好的并发处理能力
- 优化的 UI 操作性能
