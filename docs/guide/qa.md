# 常见问题

## 安装与配置

### 如何开启无障碍服务？

1. 打开 Android 设置
2. 进入"无障碍"或"辅助功能"
3. 找到 Auto.js Pro
4. 开启服务

### 应用无法启动怎么办？

- 检查无障碍服务是否已开启
- 检查应用权限是否已授予
- 重启应用或设备
- 查看错误日志

## 脚本开发

### 控件找不到怎么办？

- 检查无障碍服务是否开启
- 增加等待时间
- 使用更宽泛的查找条件
- 检查应用是否已启动
- 使用坐标点击作为备选方案

### 脚本执行失败怎么办？

- 检查权限是否授予
- 检查网络连接
- 查看错误日志
- 使用 try-catch 捕获异常
- 检查设备是否处于正确状态

### 如何调试脚本？

- 使用 `console.log()` 输出调试信息
- 使用 `toast()` 显示关键步骤
- 使用 VSCode 远程调试
- 截图保存关键状态
- 记录日志文件

## 性能优化

### 如何提升脚本性能？

- 减少不必要的截图操作
- 优化查找逻辑
- 使用异步操作
- 避免死循环
- 及时释放资源

### 内存占用过高怎么办？

- 及时释放图片对象
- 避免创建过多对象
- 使用对象池
- 优化数据结构

## 打包问题

### 打包失败怎么办？

- 检查项目配置是否正确
- 检查所有依赖是否已包含
- 查看打包日志
- 检查签名配置

### 打包后的应用无法运行？

- 检查权限配置
- 检查主入口文件
- 测试所有功能
- 查看错误日志
