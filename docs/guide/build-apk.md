# 打包为应用

## 打包步骤

### 1. 准备项目

确保您的项目结构完整，包含所有必要的文件。

### 2. 配置打包选项

在项目配置文件中设置打包选项：

- **应用名称**：设置应用的显示名称
- **包名**：设置应用的包名
- **版本号**：设置应用版本号
- **图标**：设置应用图标
- **启动图**：设置应用启动图

### 3. 执行打包

在 Auto.js Pro 中选择"打包为应用"，系统会自动：

- 编译脚本代码
- 生成 APK 文件
- 应用签名（如果配置了）

<img src="/images/build-apk-menu.cb174a12.jpg" alt="打包菜单" width="50%">

### 4. 安装测试

将生成的 APK 安装到设备上进行测试。

## 打包配置

### 基本配置

```json
{
  "name": "我的应用",
  "packageName": "com.example.myapp",
  "version": "1.0.0",
  "versionCode": 1
}
```

### 高级配置

- **代码混淆**：启用代码混淆保护
- **加密**：启用代码加密
- **权限**：配置应用权限
- **CPU 架构**：选择支持的 CPU 架构

## 注意事项

- 确保所有依赖都已包含
- 测试所有功能是否正常
- 检查权限配置是否正确
- 验证应用签名
